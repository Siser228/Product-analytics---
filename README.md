# Product analytics: анализ оттока клиентов телеком-сервиса

## Описание проекта
В рамках проекта был проведён продуктовый анализ оттока клиентов телеком-компании с целью выявления ключевых факторов churn и формирования сегмента клиентов с высоким риском оттока.

## Бизнес-задача
Снизить отток клиентов за счёт фокусировки retention-стратегий на наиболее уязвимых сегментах.

## Данные
Использован открытый датасет **Telco Customer Churn (IBM)**, содержащий данные о 7043 клиентах.

## Инструменты
- Python  
- pandas  
- matplotlib  
- seaborn  

## Ключевые инсайты
- Общий churn rate составляет ~26.5%
- Клиенты с помесячным контрактом имеют критически высокий churn (42.7%)
- Ушедшие клиенты в среднем платят больше и имеют меньший срок пользования услугами
- Сегмент высокого риска (~11.6% базы) демонстрирует churn ~69.5%

## Рекомендации
- Стимулировать переход клиентов на долгосрочные контракты
- Усилить удержание клиентов в первые 3–6 месяцев
- Таргетировать retention-офферы на сегмент высокого риска
