# Product analytics: анализ оттока клиентов телеком-сервиса

## Бизнес-задача
Снизить уровень оттока клиентов за счёт выявления ключевых факторов, влияющих на churn.

## Данные
Использован открытый датасет Telco Customer Churn (IBM), содержащий данные о 7043 клиентах телеком-компании.

## Инструменты
- Python
- pandas
- matplotlib
- seaborn

## Ключевые инсайты
- Churn rate составляет ~26.5%
- Клиенты с помесячным контрактом имеют критически высокий уровень оттока (42.7%)
- Ушедшие клиенты платят больше и имеют меньший срок пользования услугами

## Рекомендации
- Стимулировать переход на долгосрочные контракты
- Усилить удержание клиентов в первые месяцы
- Оптимизировать ценовую политику для клиентов из зоны риска

## Сегментация клиентов по риску оттока

На основе продуктовых и финансовых признаков был выделен сегмент клиентов с высоким риском оттока.  
Данный подход позволяет сфокусировать retention-стратегии на наиболее уязвимых клиентах и повысить эффективность бизнес-решений.
